# Repositório de Projetos

Este repositório contém diversos projetos de desenvolvimento, incluindo APIs e aplicações web.

## Projetos Disponíveis

### 1. API de Pagamentos
- Localização: `/API-Pagamentos`
- Descrição: API REST para gerenciamento de pagamentos, usuários e cartões de crédito
- Tecnologias: .NET 8, PostgreSQL, Entity Framework Core
- Documentação: Ver README.txt dentro da pasta do projeto

### 2. API ORDS - Validação de CPF e CEP
- Localização: `/API-ORDS`
- Descrição: API REST para validação de CPF e CEP utilizando Oracle REST Data Services (ORDS) e PL/SQL
- Tecnologias: Oracle Database, PL/SQL, ORDS
- Funcionalidades:
  - Validação de CPF
  - Consulta e validação de CEP
  - Endpoints REST para integração
- Documentação: Ver README.txt dentro da pasta do projeto

## Como Usar

Cada projeto possui sua própria documentação detalhada em seu respectivo diretório. Consulte o README.txt de cada projeto para instruções específicas de configuração e uso.
